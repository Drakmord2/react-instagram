# Reactgram
A simple clone of Instagram's UI for iOS using React Native

## Preview
![Preview](feed.gif)

## Features
- Infinite scroll;
- Image load on scroll;
- Preload blurred images;
- Pull to Refresh;

## Libraries
- React Native
- Native Base
- Styled Components
- Json Server
